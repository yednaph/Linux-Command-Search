<b> Some example usage of the `vim` command used in vim editor </b>

</b>File management</b>

:e              reload file
:q              quit
:q!             quit without saving changes
:w              write file
:w {file}       write new file
:x              write file and exit


<b> Insertion </b>:   To exit from insert mode use `Esc` or `Ctrl-C`

<b> Editing </b>

u               undo
yy              yank (copy) a line
p               paste after cursor
P               paste before cursor
<Del> or x      delete a character
dd              delete a line
